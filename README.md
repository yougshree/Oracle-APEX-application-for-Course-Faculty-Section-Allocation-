# Oracle APEX application for Course Faculty Section Allocation
## Overview

A fully functional web-based application built with **Oracle APEX** for the **CSE302 Database Systems** course at East West University. The system manages course-faculty section allocation for the CSE department — allowing administrators to assign faculty members to course sections while automatically enforcing university business rules through PL/SQL triggers.

## Dataset

Based on real **East West University CSE Department** data from Spring 2023:
- 35 full-time faculty members
- 48 courses across Core, Major Compulsory, Major Elective and Other Department categories
- Real room numbers from Main Building, AB1, AB2, AB3 and FUB buildings
- Actual semester schedule time slots
